# Graphing

This is a graphing library written in Python, using PyCairo. A working PyCairo setup is required to use the library.

# Installation

```bash
$ git clone https://github.com/GComputeNerd/Graphing-Calc.git
$ cd Graphing-Calc
$ pip install -e .
```
